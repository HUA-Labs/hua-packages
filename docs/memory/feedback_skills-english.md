스킬 파일(`.claude/skills/*/SKILL.md`)은 **영어로 작성**할 것.

**Why:** 스킬 파일은 Claude가 읽는 instruction prompt. 한국어보다 영어가 토큰 효율적이고, 모델이 영어 instruction을 더 정확하게 따름. 사용자 대화는 한국어, 스킬/코드/커밋은 영어.

**How to apply:** 새 스킬 생성 시 description, 본문 모두 영어. 기존 스킬도 수정 기회에 영어로 전환.
