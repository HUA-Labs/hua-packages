## 1. AI 관점에서 인간 감정 강도 (tier_a 상향 경향)

- gpt-5가 tier_a를 전반적으로 높게 산정 (우울기 Day 1-5 평균 a=3.86)
- 원인 가설:
  - **학습 비교 기준 차이**: LLM 학습 데이터 대부분이 중립 텍스트(문서, 코드, 뉴스). 일기 텍스트는 그 기준에서 감정 밀도가 높음
  - **미세 신호 감지**: 단어 선택, 문장 리듬, 생략, 줄임말 등 무의식적 감정 신호를 AI는 전부 읽음. 인간 본인은 담담하게 썼다고 생각하지만 AI 눈엔 신호가 풍부
  - **모델 capability와 비례**: 큰 모델일수록 미세 신호를 더 감지 → a 전반 상향 → R2 threshold 도달 빈도 증가 → sensitivity와 FP 동시 증가
- **시스템 함의**: tier_a calibration이 모델 크기에 따라 shift하면, threshold도 모델별로 조정이 필요할 수 있음

## 2. 일기 = 사회적 필터 없는 감정의 기저선

- 대화에서는 "괜찮아"로 넘기는 것을 일기에서는 솔직하게 표현
- 일기라는 매체 자체가 **unfiltered emotional state**를 담고 있음
- AI가 a를 높게 잡는 것이 과잉 해석이 아니라, 일기 텍스트의 본질적 특성일 수 있음
- **아티클 프레이밍**: "일기는 사회의 제약 없는 인간 감정 자체의 기저선을 보여준다"

## 3. 실시간 일기 vs 회고(밀려쓰기) 일기

- **실시간 일기**: 미세 감정, 전환, 분위기 변화까지 포착 → a가 세밀, emotion_flow 풍부, m이 높을 가능성
- **회고 일기**: peak memory만 남음 → 핵심 사건 + 강렬히 남은 감정 위주 → a 양극화 (확 높거나 낮거나), m 낮아질 가능성
- **peak-end rule** (Kahneman): 경험의 정점과 끝만 기억 → 밀려 쓴 일기는 자연스럽게 peak-end 필터링된 텍스트
- **SLIP 판정 영향**: 밀려 쓴 일기는 서사 동력(m) 감소, 감정 강도(a)는 잔존 기억 강도에 의존 → 실시간 vs 회고의 분석 결과가 달라질 수 있음
- **연구 주제 후보**: 작성 시점(실시간/당일/수일 후)에 따른 a, m 분포 차이와 SLIP 정확도 변화
