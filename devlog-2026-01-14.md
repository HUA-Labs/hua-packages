### 34. Carousel 무한 루프 개선

**문제:**
- 마지막 슬라이드 → 첫 슬라이드 이동 시 역방향 애니메이션

**해결 (slide transition):**
1. 슬라이드 복제: `[마지막, ...원본, 첫번째]`
2. 시작 인덱스 1로 설정 (복제된 마지막 다음)
3. 끝에서 처음으로 갈 때: 복제된 슬라이드로 자연스럽게 이동 후 점프
4. `noTransition` 상태로 점프 시 애니메이션 제거

**수정된 파일:**
- `packages/hua-ui/src/components/advanced/Carousel.tsx`

**주요 변경:**
```typescript
// 슬라이드 복제 (loop mode에서 slide transition일 때)
const slides = loop
  ? [childArray[childArray.length - 1], ...childArray, childArray[0]]
  : childArray;

// 복제 슬라이드 도달 시 실제 슬라이드로 점프
useEffect(() => {
  if (currentIndex === 0) {
    // 복제된 마지막 → 실제 마지막으로 점프
    setNoTransition(true);
    setCurrentIndex(slideCount);
  } else if (currentIndex === slideCount + 1) {
    // 복제된 첫번째 → 실제 첫번째로 점프
    setNoTransition(true);
    setCurrentIndex(1);
  }
}, [currentIndex]);
```

**인디케이터 수정:**
- `getActualIndex()` 함수로 실제 인덱스 계산
- 인디케이터 클릭 시 loop 모드면 +1 오프셋 적용
