# Dot Style Engine

`@hua-labs/dot` → `@hua-labs/hua/dot` (re-export)

> 점(dot) → 선 → 면 → UI. Tailwind 문법 영감, Tailwind 의존 제거. 웹+RN 동시 지원.

---

## Why dot?

- **hua-ui가 Tailwind에 묶여 있음** — RN 갈 수 없음
- **dot은 유틸리티 문자열을 런타임에 스타일 객체로 변환** — 플랫폼 무관
- **hua-css 전처리기에서 검증된 토큰 시스템** (18개 config, 1,100+ 토큰) 을 TS로 재구축
- dot이 있으면: hua-ui → hua-native, hue → 크로스플랫폼 SDUI

---

## Core API

### `dot()` — 유틸리티 문자열 → 스타일 객체

```typescript
import { dot } from '@hua-labs/hua/dot'

// Web → CSSProperties
const style = dot('p-4 rounded-lg bg-primary-500 text-white')
// { padding: '16px', borderRadius: '8px', backgroundColor: '#3b82f6', color: '#ffffff' }

// Transform accumulation (여러 transform → 하나의 문자열)
dot('rotate-45 scale-110 translate-x-4')
// { transform: 'rotate(45deg) scale(1.1) translateX(16px)' }

// Negative values
dot('-m-4 -translate-x-2')
// { margin: '-16px', transform: 'translateX(-8px)' }
```

**Variant 지원 (구현 완료):**

```typescript
// dark mode
dot('bg-white dark:bg-gray-900', { dark: true })
// { backgroundColor: '#111827' }

// responsive — mobile-first cascade
dot('p-4 md:p-8 lg:p-12', { breakpoint: 'lg' })
// { padding: '48px' }  (base 4 → md 8 → lg 12)

// dark + responsive 조합
dot('bg-white dark:bg-gray-900 dark:md:bg-gray-800', { dark: true, breakpoint: 'md' })
// { backgroundColor: '#1f2937' }

// state (향후 구현 예정)
dot('bg-primary-500 hover:bg-primary-600')
```

### `createDotConfig()` — 커스텀 토큰 설정

```typescript
import { createDotConfig } from '@hua-labs/hua/dot'

createDotConfig({
  theme: {
    colors: {
      brand: { 500: '#6630E6', 600: '#5525C0' },
    },
    spacing: {
      '18': '72px',
    },
    borderRadius: {
      pill: '999px',
    },
    gridCols: {
      '16': 'repeat(16, minmax(0, 1fr))',
    },
  },
  cache: true,
  cacheSize: 500,
  strictMode: false,
})
```

### `hua.config.ts` 통합

```typescript
// hua.config.ts
import { defineConfig } from '@hua-labs/hua/framework/config'

export default defineConfig({
  dot: {
    // 디자인 토큰 커스터마이징
    theme: {
      colors: {
        primary: { 500: '#71D9FF' },  // my-app cyan
      },
    },
    // 런타임 설정
    runtime: 'web',        // 'web' | 'native'
    cache: true,           // 토큰 캐시 활성화
    strictMode: false,     // 알 수 없는 토큰 에러 발생 여부
  },
})
```

## Runtime Output

### Web (CSSProperties)

```typescript
dot('p-4 flex items-center bg-blue-500')
// → {
//     padding: '16px',
//     display: 'flex',
//     alignItems: 'center',
//     backgroundColor: '#3b82f6'
//   }
```

### React Native (StyleSheet-compatible)

```typescript
dot('p-4 flex items-center bg-blue-500')
// → {
//     padding: 16,              // number (RN requirement)
//     display: 'flex',
//     alignItems: 'center',
//     backgroundColor: '#3b82f6'
//   }
```

**Key differences:**
- Web: string values (`'16px'`), kebab-case CSS properties internally
- RN: number values (`16`), no `px` suffix, camelCase only
- Variant handling differs: Web uses media queries/CSS pseudo-classes, RN uses ThemeContext/event listeners

## Relationship to hua-css

dot은 EBSe 프로젝트의 `hua-css` 전처리기에서 발전:

| Aspect | hua-css (legacy) | dot (new) |
|--------|-----------------|-----------|
| Language | JavaScript | TypeScript |
| Config | 18 JSON files | TS const + `hua.config.ts` |
| Pipeline | File I/O (CSS → CSS) | In-memory (string → object) |
| Runtime | Build-time only | Build-time + Runtime |
| Platform | Web only | Web + React Native |
| Cache | LRU (1000 entries) | WeakMap + LRU hybrid |

**재활용 대상:**
- 토큰 데이터 (colors, spacing, typography 등) → TS const로 변환
- Converter 패턴 매칭 로직 → Resolver 함수로 정리
- Breakpoint 정의 (sm/md/lg/xl/2xl) → 그대로 사용
- Gradient 방향 매핑 → 그대로 사용

## Anti-patterns

```typescript
// BAD: Tailwind className을 dot에 그대로 사용
dot('hover:bg-blue-500/20')  // ❌ opacity modifier 미지원

// BAD: arbitrary values
dot('w-[calc(100%-32px)]')   // ❌ arbitrary values 미지원

// GOOD: 정의된 토큰만 사용
dot('w-full p-4 bg-primary-500')  // ✅

// GOOD: 커스텀 값은 config에서 정의
const config = createDotConfig({
  spacing: { 18: '72px' }
})
dot('p-18')  // ✅ config에서 정의한 토큰
```
