# i18n Packages

4 packages consolidated into 3 import paths: `@hua-labs/hua/i18n`, `/loaders`, `/formatters`.

> **Important**: Translation key separator is `:` (colon), NOT `.` (dot).
> `t('namespace:key')` — e.g., `t('common:save')`

---

## @hua-labs/hua/i18n

Merges `i18n-core` + `i18n-core-zustand` + platform adapters.

### Hooks

| Export | Description |
|--------|-------------|
| `useTranslation` | Returns `t()`, `tArray()`, `tPlural()`, `getRawValue<T>()` functions |
| `useI18n` | Full i18n context (language, setLanguage, t) |
| `useLanguageChange` | Language change handler hook |
| `useZustandI18n` | Zustand-based i18n store hook |

### Providers

| Export | Description |
|--------|-------------|
| `I18nProvider` | Default i18n Provider (React context) |
| `CoreProvider` | Core Provider (internal) |
| `createLanguageProvider` | Language Provider factory |
| `createNamespaceProvider` | Namespace Provider factory |
| `createCustomLoaderProvider` | Custom loader Provider factory |

### Functions

| Export | Description |
|--------|-------------|
| `createCoreI18n` | Create i18n instance |
| `createZustandI18n` | Create Zustand-based i18n instance |
| `serverTranslate` | Server-side translation function |
| `ssrTranslate` | SSR translation function |
| `Translator` | Translator class |

### Platform Adapters

| Export | Description |
|--------|-------------|
| `webPlatformAdapter` | Web default adapter (navigator.language + window events) |
| `headlessPlatformAdapter` | No-op adapter for SSR / test / Flutter bridge |

**Types**

| Export | Description |
|--------|-------------|
| `I18nPlatformAdapter` | Platform adapter interface (for custom adapters like RN) |

### Server Subpath

`@hua-labs/i18n-core/server` (3 exports) — server-only:

```typescript
import { Translator, serverTranslate, ssrTranslate } from '@hua-labs/i18n-core/server'
```

### Common Usage Patterns

```typescript
// Client component
import { useTranslation } from '@hua-labs/hua/i18n'

function MyComponent() {
  const { t } = useTranslation()
  return <h1>{t('common:welcome')}</h1>
}
```

```typescript
// Server component (SSR)
import { getSSRTranslations } from '@hua-labs/hua/framework/server'

export default async function Page() {
  const { t } = await getSSRTranslations('ko', ['common'])
  return <h1>{t('common:welcome')}</h1>
}
```

### Cross-Platform Usage

```typescript
// React Native — custom platform adapter
import { createCoreI18n, type I18nPlatformAdapter } from '@hua-labs/hua/i18n'
import * as Localization from 'expo-localization'

const nativeAdapter: I18nPlatformAdapter = {
  getDeviceLanguage: () => Localization.locale?.slice(0, 2),
  onLanguageChange: (cb) => { /* AppState listener */ return () => {} },
}

const I18nProvider = createCoreI18n({
  platformAdapter: nativeAdapter,
  defaultLanguage: 'ko',
})
```

### Typed Namespace Hook (my-app pattern)

```typescript
// Type-safe namespace-scoped translations
import { useNamespacedTranslation } from '@/app/lib/i18n/useNamespacedTranslation'

function TermsPage() {
  const { t, tArray } = useNamespacedTranslation('terms')
  return <h1>{t('title')}</h1>  // fully typed per namespace
}
```
