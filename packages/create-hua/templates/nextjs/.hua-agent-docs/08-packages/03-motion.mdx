# Motion Package

`@hua-labs/motion-core` → `@hua-labs/hua/motion` (60+ exports re-exported)

> CSS-first motion system. Minimal JS runtime cost, automatic `prefers-reduced-motion` support.

---

## Core Motion Hooks (Most Used)

| Hook | Description | Returns |
|------|-------------|---------|
| `useFadeIn` | Fade in | `{ ref, style }` |
| `useSlideUp` | Slide up from below | `{ ref, style }` |
| `useSlideDown` | Slide down from above | `{ ref, style }` |
| `useSlideLeft` | Slide in from right | `{ ref, style }` |
| `useSlideRight` | Slide in from left | `{ ref, style }` |
| `useScaleIn` | Scale in | `{ ref, style }` |
| `useBounceIn` | Bounce in | `{ ref, style }` |
| `usePulse` | Repeating pulse | `{ ref, style }` |

```typescript
import { useFadeIn, useSlideUp } from '@hua-labs/hua/motion'

function Card() {
  const fadeIn = useFadeIn()
  return <div ref={fadeIn.ref} style={fadeIn.style}>Hello</div>
}
```
